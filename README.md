# Xash3D-iOS project

Place to SDL2/Xcode-iOS/ and use Xcode to build

Use microndk to generate game projects

### Default game libraries should be named:

* server
* client
* menu

### Default extras asset:

* extras.pak

### For mods:

* menu_<gamedir>
* server_<gamedir>
* client_<gamedir>
* extras_<gamedir>.pak

By default, gamedir used, but may be overrided with Library Suffix option
